- [x] 数据类型至少包括char类型、int类型和float类型，字符串作为可选项；

- [x] 基本运算至少包括算术运算、比较运算、自增自减运算和复合赋值运算；

- [x] **多维数组**。另外结构作为可选项；

- [x] 语言支持行注释与块注释，不要求支持编译预处理命令和多文件程序.
